

# Drishyamitra — AI Photo Management App

A premium, production-ready photo management SaaS with an Apple Photos / Google Photos aesthetic, featuring a blue/purple gradient accent palette, clean typography, and smooth micro-interactions.

## Design System
- **Theme**: Light default with dark mode toggle
- **Accent**: Blue-to-purple gradient for primary elements
- **Style**: Rounded corners, soft shadows, glassmorphism touches, smooth transitions
- **Typography**: Clean, modern spacing with clear hierarchy

## Pages & Features

### 1. Landing Page
- Navbar with "Drishyamitra" logo, navigation links (Home, Features, Login), and dark mode toggle
- Hero section with tagline "Your Intelligent AI Photo Assistant", gradient CTA buttons, and decorative illustration
- Features section highlighting core capabilities (AI Search, People Recognition, Smart Albums, Chat Assistant)
- Modern footer with links and branding

### 2. Authentication Pages
- **Login**: Centered card with email/password inputs, show/hide password toggle, gradient submit button
- **Signup**: Centered card with name, email, password fields, same clean layout
- Subtle background pattern/gradient

### 3. Main App Layout (Post-Login)
- Persistent sidebar navigation: Dashboard, Gallery, People, Chat Assistant, Upload
- Top navbar with search, dark mode toggle, profile avatar dropdown
- Responsive: sidebar collapses to icons on tablet, hamburger on mobile

### 4. Dashboard
- Overview stat cards (Total Photos, People Detected, Recent Activity, Storage Used) with gradient accents
- Recent photos grid preview (placeholder images)
- Quick upload button and welcome greeting

### 5. Gallery Page
- Search bar with filter pills (People, Date, Favorites)
- Responsive grid of photo cards with hover overlay (favorite, delete, expand)
- Full-screen modal viewer with navigation arrows
- Empty state with illustration

### 6. People Page
- Grid of person cards with avatar and name
- Click to view person's photos
- Search/filter and empty state

### 7. Chat Assistant Page
- Chat bubbles (user right, assistant left), scrollable area
- Input box with send button, typing indicator animation
- Welcome state with suggested prompts

### 8. Upload Page/Component
- Drag-and-drop zone with dashed border and icon
- Image preview thumbnails before upload
- Progress bar UI and success/error states

## Reusable Components
- Sidebar, Navbar, PhotoCard, PersonCard, ChatMessage, UploadDropzone, ModalViewer, Loading Skeletons, Empty States

## Polish
- Fade/scale animations on page transitions and cards
- Hover effects on all interactive elements
- Loading skeletons throughout
- Fully responsive (mobile, tablet, desktop)

> All data is mock/placeholder — frontend only, backend can be added later.
